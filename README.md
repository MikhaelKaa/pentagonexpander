# PentagonExpander #
Проект комплекта плат расширения для классического Пентагон 128к.  
За основу взята плата с сайта http://micklab.ru/ZX%20Spectrum/Pentagon128.htm  
Оригинальная плата подвергается минимальным доработкам, в основном все сводится к перерезанию нескольких дорожек и  
напаиванию соеденительных проводов от точек на плате Пентагона к незадействованым пинам разъема ХР3.  
Состоит из двух основных модулей:
- Плата Периферии. 
- Плата Системных доработок.
Проект открытый и не коммерческий. В случае коммерческого использования необходимо уведомить меня.

# Плата Периферии. #
- Микроконтроллер STM32 - отвечает за USB и PS2. Соеденен с CPLD через SPI.
- Микросхема CPLD EPM7128 - cодержит логику работы с шиной Пентагона.
Подключается в слот XP3 напрямую или через переходник.
Разъемы на плате:
- USB host. - мышь\клавиатруа\геймпад.
- PS2 2 шт. - мышь и клавиатрура.
- Jack3.5. - Звук.
- Разъем для подключения флоппи дисководов.
- Разъем для подключения IDE HDD.

## Запланированный функционал. ##
- Один USB host(клавиатура, мышь, gamepad на выбор).
- Два порта PS2 - клавиатура и мышь.
- Звуковой чип AY/YM.
- Переназначение буквы привода дисковода с клавиатуры. Меняет местами два провода SEL в интерфейсе Shugart.

### Второстепенное. ###
- SDcard.
- Возможно подмешивать в выходной сигнал звука спикер и звук загрузки\выгрузки на магнитофон. Отключается с клавиатуры.
- NemoIDE (нужно макетировать, частично в CPLD, частично рассыпухой).
- Управление расширенным ПЗУ.
- Доработка "Кэш на статической памяти в области ПЗУ".
- Управление расширенной оперативной памятью, например блокировка 128к.
- Управление кнопками Готека с клавиатуры. GPIO для других доработок.
- Макетное поле.

# Плата Системных доработок. #
- Микросхема CPLD EPM7064 или другая, еще не ясно.

Подключается вместо оперативной памяти, соеденения с платой Пентагона проводом МГТФ.
## Запланированный функционал. ##
- Расширение памяти до 512к\1м.
- Турбо режим.
- Чтение порта 7FFD.
